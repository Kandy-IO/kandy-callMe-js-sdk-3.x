# @kandy-io/callme-sdk

### Npm Install

#### Package name :

`$ npm install @kandy-io/callme-sdk`

#### Github url :

Install with the beta branch or a particular build with the Github URL

`$ npm install https://github.com/Kandy-IO/kandy-callMe-js-sdk#<build number or "beta"> --save`
